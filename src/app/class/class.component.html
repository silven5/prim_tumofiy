Завдання

Необхідно змінити проєкт розроблений у третій практичній роботі додавши до нього новий компонент, який призначений для роботи з абстрактними класами та файлом JSON, що зберігається у хмарному сховищі. Скрипт повинен виконувати перевірку даних отриманих з хмарного сховища. Будь-які некоректні значення повинні блокуватися.

Скласти програму з абстрактним батьківським класом і двома об'єктами – нащадками. Інформація про класи зберігається у JSON файлі. Реалізувати циклічне виведення параметрів об'єктів, використовуючи поліморфний контейнер - масив об'єктів базового класу.
